
## Instalar lo siguiente:
- Oracle XE
- Oracle Instant Client (y añadir variable de entorno y ruta al PATH)
- Oracle SQL Developer
- Node.js
- Postman

## Luego, abrir carpeta backend-convenios en la terminal y ejecutar:
```
npm install
```
